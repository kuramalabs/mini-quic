# mini-quic

A from-scratch reliable-transport-over-UDP implementation in Rust,
built as a learning path toward understanding QUIC's design.

This is NOT QUIC. No TLS, no congestion control (yet), no multiplexed
streams (yet). It's a deliberately scoped-down project working up to
those concepts one piece at a time.

See PROGRESS.md for what's actually implemented so far.
